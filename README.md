# guessTheNumberGame
🕹️ guessTheNumberGame A simple number guessing game built with HTML, CSS, and JavaScript. The player <br>
   has to guess a randomly generated number within a set range. Fun and interactive way to practice <br>
   basic JavaScript logic and DOM manipulation.  <br>
Features:  <br>
Random number generation  <br>
Input validation and feedback  <br>
Score tracking  <br>
Responsive design<br>
